Telco Customer Churn - Streamlit App

Aplicación interactiva desarrollada en **Streamlit** para el análisis exploratorio de datos (EDA) del dataset **Telco Customer Churn**.  
El objetivo del proyecto es analizar los factores que influyen en la deserción de clientes (Churn) y evaluar su impacto económico mensual.

Funcionalidades:
- Carga y visualización del dataset
- Análisis descriptivo de clientes
- Análisis de Churn por variables categóricas
- Visualización del impacto económico del Churn
- Conclusiones estratégicas para la retención de clientes
